## Feature: Todo Management

Users can create, view, update, and delete todos.

## Capabilities

### Create Todo
- User enters text
- Submits form
- Todo added to list
- Input cleared

### View Todos
- All todos displayed
- Shows text and completion status
- Visual indication of completed items

### Complete Todo
- User toggles checkbox
- Todo marked complete/incomplete
- Visual feedback (strikethrough)

### Delete Todo
- User clicks delete button
- Todo removed from list
- No confirmation (keeps example simple)

## Implementation

Minimal REST API with standard CRUD operations.
